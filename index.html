My Projects

Project 1
A Coding refactor of the marketing company Horiseon.

The aim of the project was to refactor the code of the Horiseon Marketing website without interfering with the styling of the page. The refactor aimed to increase the accessibility of the page for those with a disability or visual impairment, and to improve the readability of the code. To increase the accessibility of the code non-semantic HTML (such as div tags) were replaced with semantic HTML elements, this enables individuals using a screen reader (or other equipment) to navigate through the various sections of the page without difficulty. Images were also provided with descriptions. The readability was improved by removing unnecessary and repetitive CSS selectors. Comments were also included throughout the HTML and CSS documents, making the code easier to understand.
